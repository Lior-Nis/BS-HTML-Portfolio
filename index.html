Projects

Face Detection app

In this project, I used OpenCV to capture images, labeled them with the Labelme app, and preprocessed the data with TensorFlow's Dataset API. I split the data into train, validation, and test sets and applied data augmentation for better training. The model architecture incorporated a VGG16 base with separate branches for face classification and bounding box regression. I used a custom loss function that combined binary cross-entropy and mean squared error. After training, I evaluated the model's performance on sample test images and created a real-time app in which we can see the model in action.

Super Resolution Neural Network

In this deep learning project, I explored various CNN models for super-resolution. I fine-tuned hyperparameters and architectures, including Init, Base, Residual, Dilation, VGG pre-trained, Hybrid, and CLIP models. Bicubic upsampling significantly improved visual quality, and Residual/Dilation blocks enhanced both metrics and visuals. The report showcases my expertise in image super-resolution and deep learning, perfect for my portfolio website.
